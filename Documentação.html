Coleção de Fotos API

A API Coleção de Fotos contem os seguintes recursos:

Consulta pelo ID da foto. Metodo GET /foto/{id} 

            Exemplo:
            foto/1

Consulta pelo ASSUNTO da foto. Metodo GET /foto/assunto 

            Exemplo:
            foto/assunto?nome=batman

Consulta por ASSUNTO e COLECAO. Metodo GET /foto/consulta 

            Exemplo:
            /foto/consulta?assunto=batman&colecao=dc
